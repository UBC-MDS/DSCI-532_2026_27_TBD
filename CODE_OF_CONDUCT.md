# Code of Conduct

## Our Pledge

We as contributors and maintainers commit to fostering an open, respectful, and inclusive environment for everyone participating in this project. We value diversity of perspectives, constructive collaboration, and professional conduct.

## Our Standards

Examples of behavior that contribute to a positive environment include:

- Communicating respectfully and professionally  
- Being open to feedback and differing viewpoints  
- Providing constructive criticism  
- Focusing discussions on improving the project  
- Supporting fellow contributors  

Examples of unacceptable behavior include:

- Harassment, discrimination, or hateful conduct  
- Personal attacks or insulting language  
- Trolling or deliberately disruptive behavior  
- Sharing private information without consent  
- Any conduct inappropriate for a professional or academic setting  

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repositories  
- Issues and pull requests  
- Discussions and comments  
- Project-related communications  

It applies both online and in any context where an individual represents the project.

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct. They may take appropriate action in response to violations, including:

- Issuing a warning  
- Removing content  
- Temporarily or permanently restricting participation

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers through GitHub Issues (if appropriate) or direct communication.

All reports will be handled promptly and confidentially.
